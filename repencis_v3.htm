    How to :

  1. Pastikan HH anda telah memakai firmware DXLC1 (Optional)
  2. Masuk Recovery Mode dengan menekan [ Powe + Home + Volume Up ]
  3. Flash menggunakan CWM   Download link : http://db.tt/sIzhV4JP
  4. Setelah selesai melakukan flash, lakukan wipe data a.k.a factory reset dan wipe cache partition
  5. Reboot
  6. Setelah masuk tampilan Home Screen, dan tunggu Media Scanning sampai selesai.

Untuk Pro User dapat mengaktifkan dalvik-cache a2sd darktremor symlinking, dengan cara :

  1. Membuat partisi ext2 atau ext3 pada SDcard
  2. Menginstall BTE Pro Download Link : http://db.tt/wbRI7Su3
  3. Buka BTE Pro, Option > Preference > Input Method > Word Based
  4. Tuliskan kode berikut ini :

..............$ su (enter)

..............# a2sd cachesdreset (enter)

 

  • Buka BTE Pro lagi,
  • Tulis kode berikut ini :

..............$ su (enter)

..............# a2sd reinstall (enter)

 HH akan ter-reboot otomatis, dan lihatlah memori telepon HH anda
