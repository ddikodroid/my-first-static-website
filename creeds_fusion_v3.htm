Instructions

-Download the Rom zip File
-Copy it to root of Sdcard
-Power Off the Phone
-Boot into recovery by pressing (Vol. Key Up + Home Key + Power Key)
-Do a Data RESET
-Select Install Zip From SdCard
-Choose the rom zip
-Install
-Reboot
