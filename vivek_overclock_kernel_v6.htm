Vivek Overclock Kernel v6



Download : Vivek_-_Overclock_v6

Mirror : http://www.mediafire.com/?wmr3vmxmvjnbern

Boot.img file only : http://www.mediafire.com/?7zowupe7yl0ef96

Min : 500
Max : 832

  • extract the boot.img file from Vivek file, using WINRAR or whichever unzipper
  • place boot.img file in the root directory of the sd-card, i.e. "/sdcard/" of your SGY phone
  • also copy SGYbiCWM.zip and CWM.zip into /sdcard/
  • boot into stock ANDROID SYSTEM RECOVERY (white menu text),
  • "UPDATE" choosing the CWM.zip
  • while in CWM now (yellow menu text),
  • "install zip from sd" choosing the SGYbiCWM.zip
  • display will be: "Kernel update OK"


reboot and you are done and have a new kernel.

 

 

use  "cpu spy" app to verify which kernel you are on.

The result :
If you read "official.totoro@", then you are on a stock kernel.
If you read "vivek@.... " , you are running a vivek kernel



Vivek features many governors & schedulers !


Attention:

  • If you flash "build V6" your phone will lose the settings of all apps, flash V8.5 (OR LATER versions) instead for keeping them
